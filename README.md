# ArrestMaster для Garry's Mod DarkRP

Современная и удобная система арестов для серверов Garry's Mod DarkRP, специально разработанная для Управления Службы Безопасности (УСБ).

## Возможности

- 🚔 3D меню для ареста игроков с современным интерфейсом
- 📝 Предустановленные причины ареста с возможностью ввода своей
- ⏱️ Настраиваемое время заключения (от 1 минуты до 24 часов)
- 🏢 Автоматическое назначение камер с проверкой занятости
- 🔫 Изъятие и возврат оружия при аресте/освобождении
- 👮 Система прав на основе профессий УСБ и администраторов
- 🛡️ Расширенные настройки безопасности
- 📊 Подробное логирование всех действий
- 🔓 Досрочное освобождение с указанием причины
- 🔔 Система уведомлений
- ⚡ Оптимизированная производительность

## Установка

1. Скачайте аддон
2. Распакуйте содержимое в папку `garrysmod/addons` вашего сервера
3. Перезапустите сервер или используйте команду `lua_refresh` в консоли

## Настройка

Аддон можно настроить в файле `lua/arrestmaster/config.lua`. Все настройки разделены на категории:

### Основные настройки
```lua
DefaultJailTime = 5,           -- Стандартное время заключения в минутах
MaxJailTime = 1440,            -- Максимальное время заключения в минутах (24 часа)
MinJailTime = 1,               -- Минимальное время заключения в минутах
```

### Настройки камер
```lua
JailPositions = {
    Vector(-1620, -1000, -100),
    Vector(-1620, -1100, -100),
    -- Добавьте свои позиции камер
},
JailSpawnAng = Angle(0, 90, 0), -- Угол поворота в камерах
```

### Настройки прав доступа
```lua
AllowedJobs = {
    ["director_usb"] = true,           -- Директор УСБ
    ["operative_agent"] = true,        -- Оперативный Агент УСБ
    ["usb_officer"] = true,            -- Офицер УСБ
    ["specops_usb"] = true,            -- Группа Спец Назначения УСБ
    ["usb_pilot"] = true,              -- Пилот-Диспетчер УСБ
    ["prk_usb"] = true,                -- Рекрутированный ПРК УСБ
    ["medic_usb"] = true,              -- Медэксперт УСБ
    ["shock_usb"] = true               -- Ударный Солдат УСБ
}
```

### Настройки интерфейса
```lua
UI = {
    ArrestKey = KEY_F3,                -- Клавиша для открытия меню ареста
    ReleaseKey = KEY_F4,               -- Клавиша для открытия меню освобождения
    MenuWidth = 400,                   -- Ширина меню в пикселях
    MenuHeight = 500,                  -- Высота меню в пикселях
    MenuColor = Color(40, 40, 40, 230), -- Цвет фона меню
    TextColor = Color(255, 255, 255),  -- Цвет текста
    AccentColor = Color(178, 34, 34),  -- Акцентный цвет (красный УСБ)
    ButtonColor = Color(60, 60, 60),   -- Цвет кнопок
    ButtonHoverColor = Color(80, 80, 80) -- Цвет кнопок при наведении
}
```

### Настройки безопасности
```lua
Security = {
    MaxDistance = 100,                 -- Максимальная дистанция для ареста/освобождения
    CheckLOS = true,                   -- Проверка прямой видимости
    PreventSelfArrest = true,          -- Запретить самому себе арест
    PreventArrestingAdmins = true,     -- Запретить арест администраторов
    PreventArrestingUSB = true,        -- Запретить арест сотрудников УСБ
    RequireCuffs = false,              -- Требовать наручники для ареста
    StripWeapons = true,               -- Изымать оружие при аресте
    ReturnWeapons = true,              -- Возвращать оружие при освобождении
    SavePosition = true,               -- Сохранять позицию при аресте
    ReturnToPosition = true            -- Возвращать на позицию при освобождении
}
```

## Требования к правам

Система арестов доступна только для следующих категорий игроков:
- 👑 Администраторы и выше (admin+)
- 👮 Сотрудники УСБ следующих должностей:
  - 🎖️ Директор УСБ
  - 🔍 Оперативный Агент УСБ
  - 👨‍✈️ Офицер УСБ
  - 🎯 Группа Спец Назначения УСБ
  - ✈️ Пилот-Диспетчер УСБ
  - 🚀 Рекрутированный ПРК УСБ
  - 🏥 Медэксперт УСБ
  - ⚔️ Ударный Солдат УСБ

## Использование

### Арест игрока
1. Посмотрите на игрока, которого хотите арестовать
2. Нажмите F3 для открытия меню ареста
3. Выберите причину или введите свою
4. Установите время заключения
5. Нажмите "АРЕСТОВАТЬ" для заключения игрока

### Досрочное освобождение
1. Посмотрите на заключенного игрока
2. Нажмите F4 для открытия меню освобождения
3. Выберите причину освобождения или введите свою
4. Нажмите "ОСВОБОДИТЬ" для досрочного освобождения игрока

## Админ-команды

- `arrestmaster_unjail <id игрока>` - Принудительное освобождение игрока (только для админов)

## Система логирования

Все действия с арестами и освобождениями автоматически записываются в лог-файлы в папке `garrysmod/data/arrestmaster/logs/`. Логи создаются ежедневно в формате `YYYY-MM-DD.txt`.

### Информация в логах
- ⏰ Время и дата действия
- 📋 Тип действия (арест, освобождение, отключение игрока)
- 👤 Информация об игроках (ник, SteamID, профессия)
- 📝 Причины действий
- ⏱️ Время заключения
- 📍 Позиции игроков

### Типы логов
- Аресты игроков
- Досрочные освобождения
- Отключения заключенных игроков
- Действия администраторов
- Использование команд

## Требования

- 🎮 Garry's Mod
- 🏢 DarkRP
- 💻 Минимум 100MB свободного места для логов
